# Super Mouse Gesture

**スーパードラッグ機能付きマウスジェスチャー Chrome拡張機能**

## 概要

Super Mouse Gestureは、マウスジェスチャーとスーパードラッグ機能を提供するChrome拡張機能です。マウスの動きだけで様々なブラウザ操作やWebページ操作を効率的に実行できます。

## 主な機能

### 🖱️ マウスジェスチャー機能
マウスの右ボタンを押しながら描いた軌跡パターンで、様々なブラウザ操作を実行できます。

**対応アクション：**
- **ページ操作**: 戻る、進む、リロード、読み込み停止
- **スクロール**: ページトップへ、ページ底へ
- **ズーム**: 拡大、縮小
- **検索**: 検索バーを表示
- **タブ操作**: 新しいタブ、タブを閉じる、左側のタブを全て閉じる、右側のタブを全て閉じる
- **タブ移動**: 次のタブ、前のタブ
- **タブ管理**: 閉じたタブを復元、タブを複製、タブをピン留め、タブをミュート
- **ウィンドウ操作**: 新しいウィンドウ、シークレットウィンドウ、ウィンドウを閉じる
- **ウィンドウ制御**: 最小化、最大化、フルスクリーン切り替え

### 🎯 スーパードラッグ機能
テキスト、リンク、画像を選択してドラッグすることで、方向に応じた操作を実行できます。

**対応要素と操作：**

#### テキストドラッグ
- **検索**: Google検索、Bing検索
- **URL**: URLとして開く
- **コピー**: テキストをクリップボードにコピー

#### リンクドラッグ
- **タブ操作**: バックグラウンドタブで開く、フォアグラウンドタブで開く
- **ダウンロード**: リンク先をダウンロード
- **コピー**: リンクURLをコピー

#### 画像ドラッグ
- **タブ操作**: 新しいタブで画像を開く
- **ダウンロード**: 画像をダウンロード
- **検索**: Google画像検索
- **コピー**: 画像URLをコピー

## 技術仕様

### 開発環境
- **言語**: TypeScript
- **フレームワーク**: React（オプションページ用）
- **ビルドツール**: Webpack
- **テスト**: Jest
- **DI**: Inversify
- **Chrome Extension**: Manifest V3

## セットアップ

### 必要環境
- Node.js 16以上
- npm または yarn

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/NikawaMk2/super-mouse-gesture.git
cd super-mouse-gesture

# 依存関係をインストール
npm install
```

### 開発用ビルド

```bash
# 開発モードでビルド
npm run dev

# プロダクションビルド
npm run build
```

### テスト実行

```bash
# 全テスト実行
npm test

# テスト監視モード
npm run test:watch

# カバレッジ付きテスト
npm run test:coverage

# デバッグログ付きテスト
npm run test:debuglog
```

## 使用方法

### マウスジェスチャー
1. Webページ上で右クリックを押したまま
2. マウスを動かして軌跡を描く
3. 右クリックを離すとアクションが実行される

### スーパードラッグ
1. テキスト、リンク、または画像を選択
2. 選択した要素をドラッグ
3. ドラッグ方向に応じたアクションが実行される

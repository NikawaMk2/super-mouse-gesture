# 03. 画面設計書

## 1. 概要
本拡張機能は設定画面を持たず、Webページ上にオーバーレイ表示されるUI要素のみを持つ。
主なUI要素は「ジェスチャトレイル」と「アクション名表示」である。
アクション名表示はマウスジェスチャとスーパードラッグの両方で使用される。

## 2. UI要素詳細

### 2.1. ジェスチャトレイル (Gesture Trail)
マウスジェスチャ入力中、マウスカーソルの移動軌跡を描画する。

| 項目 | 仕様 | 備考 |
| :--- | :--- | :--- |
| **表示タイミング** | 右クリックを押下しながらマウスを移動させた時 | |
| **非表示タイミング** | 右クリックを離した時、またはジェスチャがキャンセルされた時 | |
| **描画方式** | HTML5 Canvas (全画面オーバーレイ) | `pointer-events: none` で操作を阻害しないこと |
| **色** | `#FF4081` (ビビッドピンク) | デフォルト値。視認性を確保する。 |
| **太さ** | 4px | デフォルト値。 |
| **スタイル** | 実線、角は丸く (lineJoin: round, lineCap: round) | |

### 2.2. アクション名表示 (Action Status Info)
入力中のジェスチャまたはスーパードラッグ操作に対応するアクション名を画面中央またはカーソル付近に表示する。

#### 2.2.1. マウスジェスチャ用アクション名表示

| 項目 | 仕様 | 備考 |
| :--- | :--- | :--- |
| **表示内容** | 認識されたジェスチャに対応するアクション名 (例: "新規タブ", "戻る") | ジェスチャ未定義の場合は表示しない、または"Unknown"等は表示しない |
| **表示位置** | 画面中央 (Center) | 視認性を優先。将来的にはカーソル追従も検討。 |
| **スタイル** | ・背景色: 半透明黒 (`rgba(0, 0, 0, 0.7)`)<br>・文字色: 白 (`#FFFFFF`)<br>・フォントサイズ: 24px<br>・パディング: 10px 20px<br>・角丸: 4px | CSSで定義 |
| **表示タイミング** | 有効なジェスチャとして認識された瞬間 | |
| **非表示タイミング** | 右クリックを離した後、即時または短時間フェードアウト | |

#### 2.2.2. スーパードラッグ用アクション名表示

| 項目 | 仕様 | 備考 |
| :--- | :--- | :--- |
| **表示内容** | ドラッグされたデータタイプに対応するアクション名 (例: "リンクを開く", "テキスト検索") | ドラッグデータが認識できない場合は表示しない |
| **表示位置** | 画面中央 (Center) | 視認性を優先。将来的にはカーソル追従も検討。 |
| **スタイル** | ・背景色: 半透明黒 (`rgba(0, 0, 0, 0.7)`)<br>・文字色: 白 (`#FFFFFF`)<br>・フォントサイズ: 24px<br>・パディング: 10px 20px<br>・角丸: 4px | ジェスチャ用と同じスタイル |
| **表示タイミング** | ドラッグ開始時、ドラッグされたデータタイプ（リンクまたはテキスト）が認識された瞬間 | |
| **非表示タイミング** | ドロップ後、即時または短時間フェードアウト | |
| **トレイル表示** | なし | スーパードラッグではトレイルは表示しない |

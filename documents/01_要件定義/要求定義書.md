# 要求定義書

## 解決したい課題

以下の操作をChrome上で行えるようにしたい
- スーパードラッグによる検索・ページ遷移
- マウスジェスチャによるページ・タブ操作

## 利用したい操作

### スーパードラッグ

#### 1. キーワード検索

ページ上でテキストを選択した状態で上下左右のいずれか1方向にドラッグさせると、選択したキーワードで検索を行い、その結果を別タブで表示する。
(検索エンジンは既定のものを使用)

#### 2. 別ページへのリンク遷移

ページ上のリンクを上下左右のいずれか1方向にドラッグさせると、選択したリンクをを別タブで表示する。

### マウスジェスチャ

| 方向 | アクション |
| --- | --- |
| 上 | 上へスクロール |
| 下 | 下へスクロール |
| 左 | 戻る |
| 右 | 進む |
| 上左 | 前のタブを選択 |
| 上右 | 次のタブを選択 |
| 下左 | このタブを閉じて左のタブを選択 |
| 下右 | このタブを閉じて右のタブを選択 |
| 左上 | ページ拡大 |
| 左下 | ページ縮小 |
| 右上 | 新規タブを開く |
| 右下 | フルスクリーン表示 |
| 上下 | ページの一番下へ |
| 下上 | ページの一番上へ |
| 左右 | 再読み込み |
| 右左 | タブを復元する |

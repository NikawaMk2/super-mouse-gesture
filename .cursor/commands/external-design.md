# external-design

以下の流れで外部設計書を作成すること。
1. 指定された要件定義書を確認し、内容を理解する。要件定義書が指定されていない場合、その旨をユーザに通知してタスクを終了する。
2. 不明点がある場合、ユーザに確認を行う。不明点が全て解消されたら以降のタスクを実行する。
3. 確認した内容から外部設計書を`documents/02_外部設計`配下に作成する。作成するファイルは以下とする。
- 01_概要.md
- 02_システム構成図.md
- 03_画面設計書.md
- 04_機能仕様書.md
- 05_データ設計書.md
- 06_インターフェース設計書.md
- 07_Manifest定義書.md
- 08_非機能要件設計書.md
- 09_ログ出力方針設計書.md
上記以外に作成する必要のある内容が存在する場合、ユーザに追加するファイルとその内容(概要)を提示して作成許可を得ること。
